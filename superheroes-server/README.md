## Description

Superhero DB back-end implements simple CRUD for Hero entities. Properly handles images, saving them on server's disk.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
